저 URL은 무엇인가?
Software Path(ISV Path) 단계별 요구사항/혜택 페이지입니다. AIGENDRUG가 AWS 파트너로서 어떤 단계에 있고, 다음 단계로 가려면 무엇이 필요한지 보여줍니다.

FTR은 전체 절차의 일부일 뿐입니다
Software Path는 4단계 구조이고, FTR은 3단계(Validated)로 진입하는 필수 조건입니다:

단계	요구사항	주요 혜택
Enrolled	Software Path 등록	트레이닝, AWS Blueprint, Marketplace 접근
Confirmed	APN 연회비 $2,500 납부	Promotional Credits, Marketing Central
Validated ← FTR 필요	FTR 완료	AWS Partner Badge, Partner Solution Finder 등재, ISV Accelerate, ACE Referrals, Qualified Software 배지
Differentiated	Competency / MSP / Service Ready / WAPP 중 하나 취득	MDF, Migration Acceleration Program, APN Blog post
그래서 FTR만 하면 끝이냐?
목표가 뭐냐에 따라 다릅니다.

AWS와 공동 영업(Co-sell) + ISV Accelerate 참여 → FTR(Validated) 완료하면 됨
AWS Competency / Service Ready 취득 → FTR 이후 추가 심사 필요 (Differentiated 단계)
Partner Solution Finder 등재 + Partner Badge → FTR(Validated)로 가능
현재 AIGENDRUG 상황에서 FTR 완료 = Validated 단계 진입이고, 이게 가장 먼저 해야 할 목표입니다.
