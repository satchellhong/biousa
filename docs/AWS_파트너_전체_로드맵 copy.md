### Validated 단계 주요 프로그램 상세

#### ISV Accelerate — 가장 중요한 혜택
> AWS 영업팀과 공동으로 딜을 추진하는 Co-sell 프로그램

**핵심 수치**: 파트너의 51%가 Co-sell 결과로 더 높은 매출 성장, 65%가 더 빠른 딜 클로저 경험

**혜택**:
- AWS Account Manager들의 솔루션 추천 풀(Pool)에 등재
- AWS Marketplace Private Offer 경유 딜 성사 시 AWS AM에게 인센티브 제공 (적극 추천 동기 부여)
- Co-sell 전용 교육, 이벤트, 베스트 프랙티스
- ISV 전용 펀딩 및 Marketplace Private Offer Promotion Program 인센티브

**신청 조건**: FTR(Validated) 완료 + AWS Marketplace 등록  
**신청 방법**: Partner Central → Programs → ISV Accelerate

---

#### ACE (APN Customer Engagements) — Co-sell 파이프라인 관리
> AWS와 함께 고객 기회를 공동 관리하는 플랫폼

**혜택**:
- AWS와 공동 영업 기회 발굴 및 파이프라인 공유
- AWS 영업팀으로부터 Referral(고객 추천) 수신
- 신규 기회 창출 시 금전적 인센티브
- MDF 및 POA 펀딩 접근권

**활용 방법**: Partner Central → Sell → ACE Pipeline Manager

---

#### POA (Partner Opportunity Acceleration) Funding
> 딜 클로저 가속을 위한 자금 지원

**지원 내용**: 마케팅 활동, 고객 PoC, 기술 지원 비용 지원  
**신청 방법**: Partner Central → Funding → POA 신청 (ACE 기회와 연동)

---

#### AWS Global Sponsorship
> AWS re:Invent, AWS Summit 등 대형 이벤트 스폰서십

**규모**: 연간 50만+ 고객/잠재고객 도달, 전 세계 이벤트  
**적합한 경우**: 브랜드 인지도 확대, 글로벌 시장 진출  
**신청**: AWS 파트너 담당자를 통해 스폰서십 패키지 문의

## STAGE 4 — Differentiated (차별화)

### 요구사항 (아래 중 하나 취득)
1. **AWS Competency** — 산업/기술 분야 전문성 최고 등급 인증
2. **AWS MSP Program** — 종합 매니지드 서비스 파트너 인증
3. **AWS Service Delivery** — 특정 AWS 서비스 전문 납품 능력 인증
4. **AWS Service Ready** — 특정 AWS 서비스와의 통합/호환성 인증
5. **AWS Well-Architected Partner Program (WAPP)** — WAFR 수행 능력 인증

### Differentiated 추가 혜택

| 혜택 | 설명 |
|---|---|
| **Committed MDF** | 확정된 Marketing Development Funds (금액 보장) |
| **Migration Acceleration Program** | 고객 마이그레이션 프로젝트 AWS 투자 지원 |
| **Enhanced AWS Partner Badge** | Validated보다 높은 등급 배지 |
| **Partner Solution Finder 우선 노출** | 검색 결과 상위 랭킹 |
| **AWS 전문가 기술 세션** | AWS 스페셜리스트 팀과 직접 기술 협업 세션 |
| **APN Blog Post** | AWS 공식 블로그에 파트너 성공 사례 게재 |
| **지정 분야별 추가 펀딩** | Competency 종류에 따른 추가 인센티브 |
| **AWS GameDay Partner League** | AWS GameDay 이벤트 파트너 리그 참여 |

### Differentiated 단계 주요 프로그램 상세

#### AWS Competency — 최고 수준의 차별화
> 특정 산업 또는 기술 분야에서 검증된 전문성과 고객 성공 사례 보유 파트너

**주요 Competency 종류** (AIGENDRUG 관련성 높은 것):
| Competency | 적합 조건 |
|---|---|
| AI/ML Software | AI/ML 솔루션 제공 |
| Healthcare | 의료/헬스케어 분야 |
| Life Sciences | 생명과학/바이오 분야 |
| Data & Analytics | 데이터 분석 플랫폼 |
| Security Software | 보안 솔루션 |

**취득 절차**: FTR 이후 별도 심사 (고객 레퍼런스 + 기술 검증 포함)  
**소요 기간**: 수개월  
**링크**: [AWS Competency Programs](https://aws.amazon.com/partners/offerings/)

---

#### AWS MSP Program — 매니지드 서비스 전문 파트너
> 클라우드 계획, 마이그레이션, 운영, 최적화 전 주기 서비스 제공 능력 인증

**특징**:
- 제3자 감사(third-party audit)를 통한 엄격한 검증
- 고객 성장률 3배 향상 효과 (2024 AWS 고객 설문)
- 52% 고객이 MSP와 협업 필요성 표명

**적합한 경우**: 고객 AWS 환경 관리/운영 대행 서비스 제공 시  
**링크**: [AWS MSP Program](https://aws.amazon.com/partners/programs/msp/)

---

#### AWS Well-Architected Partner Program (WAPP)
> 고객 워크로드에 대해 Well-Architected Framework Review를 수행할 수 있는 파트너 인증

**혜택**:
- WAFR 수행으로 고객 신뢰 구축 및 신규 프로젝트 창출
- AWS 파트너 배지에 Well-Architected 전문성 표시
- **WAFR 완료 시 FTR 단축 경로 활용 가능** (역으로 WAPP이 FTR에도 도움)
- 고객 ARR의 최대 10% AWS 크레딧 보상 (WAFR 성과 기반)

**신청 조건**: FTR(Validated) 완료 + AWS SA-Professional 자격증 보유자 1명 이상  
**링크**: [WAPP](https://aws.amazon.com/partners/programs/well-architected/)

---

#### ISV Workload Migration Program (WMP)
> 고객을 SaaS 솔루션으로 마이그레이션 시키는 ISV 파트너 지원 프로그램

**혜택**:
- 고객 마이그레이션 비용 상쇄를 위한 AWS Promotional Credits 제공
- Go-to-market 지원 및 기술 인에이블먼트
- 마이그레이션 성공 시 펀딩

**적합한 경우**: 기존 On-prem 또는 타 클라우드 고객을 AWS 기반 SaaS로 전환 유도  
**링크**: [ISV WMP](https://aws.amazon.com/partners/programs/isv-workload-migration/)

---

#### APN Ambassador Program
> AWS 파트너 커뮤니티 내 기술 리더십 인정 프로그램

**대상**: 파트너사 내에서 AWS 전문성을 주도하는 기술 리더 개인  
**혜택**: 커뮤니티 활동, AWS 이벤트 초청, 글로벌 네트워킹  
**링크**: [APN Ambassador](https://aws.amazon.com/partners/ambassadors/)

---

#### Migration Acceleration Program (MAP)
> 대규모 엔터프라이즈 고객 클라우드 마이그레이션 지원 프로그램

**3단계 프레임워크**: Assess → Mobilize → Migrate & Modernize  
**투자**: AWS가 마이그레이션 비용의 일부를 직접 투자  
**적합한 경우**: 대형 고객의 레거시 시스템 마이그레이션 프로젝트 수주 시  
**링크**: [MAP](https://aws.amazon.com/migration-acceleration-program/)

```
현재                   3~6개월                  6~12개월              12개월+
  │                       │                          │                    │
  ▼                       ▼                          ▼                    ▼

                    등재
                    ISV Accelerate 신청
```

### 단계별 우선순위

**지금 (FTR Paused 기간 활용)**:
1. 체크리스트 46개 항목 이행 → [FTR_상세_절차.md](./FTR_상세_절차.md)
2. Security Hub + CIS Benchmark 활성화
3. Architecture Diagram 작성
4. Enrolled 혜택 중 바로 활용 가능한 것 사용 (트레이닝, SaaS Factory 등)

**FTR 재개 즉시**:
5. Self-Assessment 엑셀 + CIS 리포트 + Diagram 제출
6. FTR 승인 후 ISV Accelerate 즉시 신청

**Validated 달성 후**:
7. ACE 파이프라인 활성화 → Referral 수신 시작
8. AIGENDRUG 솔루션 성격에 맞는 Competency 방향 결정
   (AI/ML, Life Sciences, Healthcare 중 선택)
